# 冰箱救星：剩食變美食 AI 幫你

## 專案概述

本專案是一個基於 AI 的剩餘食材食譜推薦系統，旨在解決使用者冰箱中剩餘食材的應用問題。透過上傳食材照片，系統會自動識別食材並推薦適合的料理食譜。

## 核心功能

### 1. 食材識別
- 支援多張圖片批次上傳
- 使用 Google Vision API 進行 AI 食材識別
- 手動編輯和調整食材清單

### 2. RAG 食譜推薦
- 基於檢索增強生成技術
- 根據食材吻合度推薦食譜
- 提供替代方案建議

### 3. 使用者體驗
- 響應式設計，支援多種裝置
- 直觀的操作介面
- 食譜詳情和步驟展示

## 技術架構

### 前端技術
- React 18 + TypeScript
- Tailwind CSS
- React Query
- React Router

### 後端技術
- Python Flask
- SQLAlchemy (ORM)
- ChromaDB (向量資料庫)
- PostgreSQL

### AI 服務
- Google Vision API (食材識別)
- OpenAI GPT API (食譜生成)
- Sentence-BERT (文本嵌入)
- LangChain (RAG 框架)

## 使用方式

1. 上傳冰箱食材照片
2. 確認 AI 識別的食材清單
3. 選擇飲食偏好和烹飪時間
4. 查看推薦的食譜
5. 按照步驟製作料理
